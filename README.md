# Department_website

HTML 4.01 Transitional

Student Name: Shumaila
Program: BS Mathematics

--------------------------------------------------

Project Description:

This project is a multi-page Department Website created using
HTML 4.01 Transitional only.

The website represents the Department of Mathematics and includes
a professional layout, navigation menu with icons, structured tables,
lists, images, and a complete contact form.

No HTML5 tags or external CSS were used as per assignment instructions.
